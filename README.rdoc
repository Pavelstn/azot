== Azot витрина для партнерских программ.

=== Назначение
Витрина для различных партнерских программ, в том числе для продажи купонов, но возможна работа и с обычными товарами.
В дальнейшем планиуется добавить функционал до полноченного интернет магазина

* Использует usermanage https://github.com/Pavelstn/usermanage  - Модуль для управления учетными данными пользователей.
